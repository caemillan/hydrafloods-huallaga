Hydrafloods for Huallaga river¶

Este repositorio muestra el mapeo de inundaciones para un tramo del ro Huallaga en Tingo Mara, aplicando los algoritmos de la aplicacin Hydrologic Remote Sensing Analysis for Floods (o HYDRAFloods).

"Edge Otsu" algorithm¶

La aplicacin HYDRAFloods incluye dentro de su modulo de umbrales 3 algoritmos genericos de mapeo de agua superficial:

  • Edge Otsu: hydrafloods.edge_otsu (Donchyts et al., 2016; Markert et al., 2020)
  • Bmax Otsu: hydrafloods.bmax_otsu (Cao et al., 2019; Markert et al., 2020)
  • Extensin de KMeans: hydrafloods.kmeans_extent(Chang et al., 2020)
In [ ]:
# apply a water mapping function to the S1 dataset
# this applies the "Edge Otsu" algorithm from https://doi.org/10.3390/rs12152469
water_imgs = s1.apply_func(
    hf.thresholding.edge_otsu,
    initial_threshold=-14,
    thresh_no_data=-20,
    edge_buffer=300
)
